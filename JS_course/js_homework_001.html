Assigement:

Write loop that will write to console all the numbers between 6 and 178 (inclusive) which are dividable by 7 but not with 3.
For example: 21 no, but 28 yes.

Solution:

              
                for (let i = 6; i <= 178; i++) {
                    if (i % 7 === 0 && i % 3 !== 0) {
                      console.log(i);
                    }  
                }
